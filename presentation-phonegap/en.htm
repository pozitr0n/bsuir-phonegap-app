How it all begin?

PhoneGap

It's a solution with an open source framework for building cross-platform mobile applications using a modern, standards-based Web technologies.

Official page

Supported platforms

  • iPhone /iPhone 3G/3GS and newer
  • Android
  • OS 4.6-4.7/OS 5.x
  • OS 6.0+
  • WebOS
  • WP7
  • Symbian and Bada

Main!
What you need to remember.

This framework intended only for emergency situations where an urgent need to write an application for mobile systems and when there are no people who know their native languages for mobile systems, such as Java for Android OS or Objective C for iOS.

How to compile?

				npm install -g phonegap
				phonegap create myApp
				phonegap remote build android
				npm install -g cordova
				cordova plugin add
				org.apache.cordova.media
