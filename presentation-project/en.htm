The Advantages

  • Easy to learn
  • Speed
  • Cross-Platform
  • Each front-end developer'll be able to realize any application for a very short period of time, not knowing the native languages
  • Time!

The Shortcomings

  • It turns out very large code. If you write in Java, it will be much less
  • Internal defects in the framework
  • There is a complexity to support older operating mobile systems
  • Time!
